ManifoldAPIs

This package allows users to interface with Manifold through the API using the Julia programming language. Navigate the panel on the left to find more information.

Installation

  1. Add registry to package manager via
using Pkg
pkg"registry add https://github.com/itsdfish/Registry.jl"
  1. Add package to environment via 
using Pkg 
Pkg.add("ManifoldAPIs")

or via package mode by entering ] followed by

add ManifoldAPIs
